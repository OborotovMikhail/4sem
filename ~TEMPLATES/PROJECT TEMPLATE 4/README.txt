PROJECT TEMPLATE DESCRIPTION:
This template has organized directories and includes OpenCV and SFML


DESCRIPTION FOR THE FOLDERS IN THIS DIRECTORY:

"FILES TO COPY"
Files from this folder should be copied to a new project's working directory

"TEMPLATE ZIP"
ZIP inside this folder is a VisualStudio project template
It should be copied to the VisualStudio's "ProjectTemplates" folder
By default on Windows this folder seems to be:
Drive:\documents\Visual Studio 2019\Templates\ProjectTemplates (This is different from the VisualStudio's installation folder!)

"TEMPLATE_mk?"
This is a VisualStudio project from which the template is created
"?" is the template version
